Adding a Tool

Step One: What shape is your brush and will it have falloff?

        You must first determine the end product of your brush, because there may be a built in behavior for the brush shape you require. If the shape is not provided in the libphoto library, you will have to write a mask behavior class for the required shape. Your mask behavior must have a maskPattern() function that returns a float between 0-1(0 represents no color change).


Step Two: Create your Tool Subclass

        You will need to create a subclass that inherits from Tool.h. It will have a constructor that takes in 2 integer parameters, bufferHeight and bufferWidth.
  • Member Variables to Set
    • m_bufferHeight = bufferHeight
    • m_bufferWidth = bufferWidth
    • m_toolSize = 1.0
    • m_width = m_defaultWidth (set the defaultWidth beforehand)
    • m_height = m_defaultHeight (set the defaultHeight beforehand)
    • m_maskSize = m_defaultMaskSize (set the defaultMaskSize beforehand)
    • Set the other member variables to your desired value
  • Use the setMaskBehavior to set the behavior of your choice
  • Call the makeMask function

Step Three: Move the Tool to the App

        You will need to add to the “tools” array in the desired App’s header file. Then initialize the tool in the constructor of the App. (tools[n] = new ToolClass(bufferHeight, bufferWidth)). To allow the user to toggle to your new tool, add a new radio button under the Glui_Panel toolPanel. Lastly, in the destructor of the App, free the memory for the new tool you just created. (delete tools[n])
