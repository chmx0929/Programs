Adding a Kernel

Back to Filter

Step One: Determine the default size of your Kernel

        If you want a faster filter, a smaller kernel will be a better choice.


Step Two: Create your Kernel Class

        The constructor of your kernel should take an integer parameter. Assign the parameter to the member variable m_kernelSize. Then call the makeKernel() function.


Step Three: Create the makeKernel() Function

        This function should not return anything. It should initialize a new float array to the member variable m_kernel. Then assign float values to the kernel. (0 represents the pixels that wont affect the center pixel, the larger the number the larger the affect on the center pixel, floats maybe negative or positive)
