Adding a Filter

Step One: Determine if your Filter is a Convolution Filter

        If your filter is a convolution filter, it will require a Kernel. Go here to learn how to add a Kernel.


Step Two: Create the Filter Class

        If the filter is a convolution filter and will be used as a brush tool, it must have a getModifiedColor function; this function returns a ColorData. If your convolution filter takes parameters other than the size of the kernel, you may need to make another function to set that parameter. The constructor for a filter requires you to set its kernelSize and other parameters, kernelSize must be an odd number. Then initialize your member variable, m_kernel, by calling your new kernel's constructor. If you wan't the users to be able to change the parameters of your filter, you will have to make a changeKernel() function that sets the parameters and calls makeKernel() from your new Kernel class. The filter will need a getColor function if it is a convolution filter. This function takes 2 integer parameters, the position of the current pixel of the pixelbuffer. The functin will need to loop through your new kernel. For each float in the kernel, you multiply it with the current pixel's RGB values. For each product, add them to a variable red, green, and blue. These variables will be used to make your new color. If the final sums of red, green, and blue are greater than 1, you will have to calculate a color factor by adding all the floats of the kernel. Divide the new RGB values by that color factor to get a float less than one, but greater than 0. The RGB values will be your new color. Lastly, create an applyFilter() function that takes in a pixelBuffer that it will modify. Loop through each pixel of that pixelBuffer and call setPixel with the getColor functin you just created.


Step Three: Add the new Filter to your App

        First create a variable for your new filter in the header file of your app. Then initialize the Filter in the constructor of the app. Finally add the neccessary functions to the GLUI to allow the user to apply the filter.
